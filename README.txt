######################################## BS|Base Communication ########################################
-It's a console-based software for internal secured communication of Blueprint System (BS).
-The tool use a specific set of algorithms to encrypt messages and transmit them securely using Bluetooth (Initially, but it could be any network control, eg.. SSH).
-The Code should be not modified or reused without a valid permission.
-This tool is available only for close-distance devices at this moment due to the constraint of Bleutooth.
